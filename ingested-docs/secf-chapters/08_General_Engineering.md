| Description: 
Foundational concepts in mathematics, science and engineering and their application. |  |  |  |  |
| Why it matters: 
Systems Engineering is performed in a technical scientific environment and as a result, a good understanding of mathematics, science coupled with a sound appreciation of core engineering principles is a critical foundation for effective Systems Engineering. Without this, systems engineers cannot communicate effectively and
efficiently with engineers from other domains. |  |  |  |  |
