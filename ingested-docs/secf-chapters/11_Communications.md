| COMPETENCY AREA - PROFESSIONAL: COMMUNICATIONS |  | SELF-ASSESSMENT COMPETENCE LEVEL :  |  | UNASSESSED |
| Description:
The dynamic process of transmitting or exchanging information using various principles such as verbal, speech, body-language, signals, behavior, writing, audio, video,
graphics, language, etc. Communication includes all interactions between individuals, individuals and groups or between different groups. |  |  |  |  |
| Why it matters:
Communication plays a fundamental role in all facets of business within an organization, in order to: transfer information between individuals and groups to develop a
common understanding and build and maintain relationships and other intangible benefits. Ineffective communication has been identified as the root cause of problems
on projects. |  |  |  |  |
