| COMPETENCY AREA - TECHNICAL: INTERFACES |  | SELF-ASSESSMENT COMPETENCE LEVEL :  |  | UNASSESSED |
| Description:
Interfaces occur where system elements interact, for example human, mechanical, electrical, thermal, data, etc. Interface Management comprises the identification,
definition and control of interactions across system or system element boundaries. |  |  |  |  |
| Why it matters:
Poor interface definition and management can result in incompatible system elements (either internal to the system or between the system and its environment) which
may ultimately result in system failure or project overrun. |  |  |  |  |
