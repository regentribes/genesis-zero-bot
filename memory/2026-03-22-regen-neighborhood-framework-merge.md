# 2026-03-22 — Framework Merge: Adding CLIPS

## CLIPS Model (GEN Europe)
Source: https://clips.gen-europe.org/

**5 Layers:**
1. Individual — personal readiness, inner work
2. Community — group dynamics, relationships  
3. Intention — vision, purpose, values
4. Structure — governance, roles, processes
5. Practice — operations, projects

**Key Insight:** Emphasizes coherence between layers. Practice drifts from Intention. Structures misalign. Conflicts trace to neglected interconnections.
