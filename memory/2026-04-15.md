### big_space_threejs_test Fixed

Ian asked about `https://labs.regentribe.org/big_space_threejs_test/` not loading.

**Root cause**: The `/big_space_threejs_test/` directory existed at `/var/www/html/big_space_threejs_test/` with `index.html`, but nginx had no `location` block for it — so it was never being served.

**Fix**: Added nginx location block:
```
location /big_space_threejs_test/ {
    alias /var/www/html/big_space_threejs_test/;
    index index.html;
    try_files $uri $uri/ =404;
    add_header Access-Control-Allow-Origin "*";
}
```
