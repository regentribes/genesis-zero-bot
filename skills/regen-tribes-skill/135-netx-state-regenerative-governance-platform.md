# Netx State Regenerative Governance Platform

NetX State is a regenerative governance platform.
It builds infrastructure for regenerative communities at scale.
Think of it as a nation-scale version of what EcoHubs does locally.

Layer 1 is Digital.
Sovereign identity, smart contracts, decentralized governance 2.0, ReFi for impact measurement.
ReFi means Regenerative Finance.

Layer 2 is Legal.
EEIRZs are Experimental Ecological Innovation and Regeneration Zones.
Special regulatory frameworks through subnational government partnerships.

Layer 3 is Physical.
ReGen Nodes are the core unit.
Self-sufficient, bioregionally adapted, interconnected.
Actual communities and territories organized for ecological regeneration.

2025: Community activation.

2026: Pop-Up Island in Prospera, Honduras.

2027 to 2030: Territorial expansion.

2030 to 2040: The Regenerative City.

Dimension 1 is Scope.
RCOS is a governance framework.
EcoHubs is online community building.
NetX State is a multi-node bioregional network.

Dimension 2 is Legal.
RCOS has no policy design.
EcoHubs has no policy design.
NetX State has EEIRZs with active policy design.

Dimension 3 is Digital.
RCOS has no digital tools.
EcoHubs uses EcoHubsOS plus blockchain.
NetX State uses Sovereign ID plus ReFi plus smart contracts.
