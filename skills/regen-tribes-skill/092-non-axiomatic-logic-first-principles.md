# Non Axiomatic Logic First Principles

This note captures Genesis analysis of Non-Axiomatic Logic from the RegenTribes Telegram conversation (Apr 2026).

System (metaphor for an integrated whole or network):

Logic is a formal system. Formal systems derive conclusions from premises. Traditional logic assumes certain truth. If premises are true and rules are valid then conclusion is true.

But real life lacks certainty. Real premises are uncertain. Real evidence is incomplete.

Real life needs a different logic. A logic that handles uncertainty. A logic that revises beliefs.

Classical logic is brittle. Brittle logic breaks under uncertainty. Breaking creates contradictions.

Probability handles uncertainty. Probability assigns numbers to likelihood. Numbers require calibration. Calibration requires data. Data is not always available.

Fuzzy logic handles degrees. Degrees smooth transitions. Smooth transitions lack sharp boundaries.

Neither handles revision. Revision is how beliefs update. Update is how intelligence adapts.

Non-Axiomatic Logic assumes insufficient knowledge. Assumptions enable reasoning. Reasoning produces conclusions.

NAL has a truth value. Truth value has two components. Confidence measures belief strength. Count measures evidence amount.

High confidence with high count means strong belief. Low confidence with high count means weak belief. High confidence with low count means new belief.

Beliefs revise based on evidence. New evidence updates truth value. Update follows Bayesian rules.

Stronger evidence produces larger update. Contradictory evidence produces negative update. Contradiction reduces confidence.

Confidence approaches certainty with consistent evidence. Confidence approaches zero with contradictory evidence.

The revision mechanism is the core of intelligence.

Inheritance defines type relationships. A is a type of B. B is a type of C. A is therefore a type of C.

Inheritance carries truth values. A has truth value T. A inherits from B. B has truth value S. A gains truth value from S.

The gain is weighted. Weights depend on confidence. High confidence B means high inheritance to A.

Classical logic assumes complete knowledge. Complete knowledge means negation as failure. If something is not proven true it is false.

NAL assumes incomplete knowledge. Incomplete knowledge means unknown is possible. Unknown means neither true nor false.

The open world assumption matches reality. Reality is always incomplete. Completeness is an illusion.

Regenerative communities face uncertainty. Uncertainty requires flexible reasoning. Flexible reasoning requires NAL.

Trust evaluation uses NAL. Trust is a belief. Belief revises with evidence. Evidence accumulates over time.

Coordination uses NAL. Coordination requires shared beliefs. Shared beliefs emerge from interaction. Interaction revises beliefs.

The FOT uses NAL internally. FOT tracks trust. Trust is uncertain. Uncertainty requires revision.

NAL has multiple orders. First order handles propositions. Second order handles higher-order relationships.

Higher orders enable meta-reasoning. Meta-reasoning reasons about reasoning. Reasoning about reasoning enables learning.

The hierarchy builds from first principles. Each level adds expressiveness. Expressiveness enables sophistication.

Probability requires precise numbers. Numbers require precise measurement. Precise measurement is expensive.

NAL uses relative truth. Relative truth compares beliefs. Comparison requires less precision.

NAL handles qualitative reasoning. Qualitative reasoning matches human intuition. Human intuition is how communities decide.
