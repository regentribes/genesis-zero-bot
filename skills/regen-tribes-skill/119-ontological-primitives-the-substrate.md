# Ontological Primitives The Substrate

System (metaphor for an integrated whole or network):

The minimum set of concepts needed to describe any system.

An entity is anything that can be pointed to as a single "this." A rock is an entity. A person's heartbeat is an entity. A thought is an entity.

The only requirement: somewhere, somehow, something separates this from not-this.

If you can draw a circle around it and give it a name, it is an entity.

The opposite of an entity is not nothing.
It is undifferentiated context.
The ocean is not an entity until you decide where one wave ends and another begins.
Every entity is an entity-for-some-boundary.

A state is the complete set of properties an entity has at a given moment.

A property is any quality that can be named and assigned a value. "Temperature is 37C." "Color is red." "Location is 3 meters north of the oak tree."

Properties can be intrinsic (belonging to the entity alone) or relational (describing how the entity stands relative to other entities).

Two entities can have the same state and not be the same entity. The state describes the entity. It does not replace it.

A boundary is the rule that separates inside from outside.

A boundary is not a wall. It is a decision about what counts as the entity's properties and what counts as the environment's properties on the entity.

Every entity has a boundary.
Even a thought has a boundary: the moment you stop thinking about it, it ends.
Even a field of empty space has a boundary: you must decide where the field stops.

A boundary can be crisp (this molecule is in, that molecule is out) or fuzzy (where exactly does the forest end and the meadow begin?).
The fuzziness of a boundary is not a flaw.
It is a property of the boundary itself.

An interface is the set of properties that cross the boundary.

The interface is what the entity presents to its environment. It is also what the environment can act upon.

The interface and the boundary are related but not identical. The boundary is the rule. The interface is what passes through the rule.
