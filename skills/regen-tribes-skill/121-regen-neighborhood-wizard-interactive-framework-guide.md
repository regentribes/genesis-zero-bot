# Regen Neighborhood Wizard Interactive Framework Guide

A Regen Neighborhood Wizard turns the whole framework stack into an interactive guided experience.

Where should this live?
- A standalone web app (HTML/CSS/JS, no backend needed for v1)
- A page on the existing regen-viz site
- Integrated into the Genesis bot experience

What should the output look like?
- A downloadable PDF project plan
- A printable markdown checklist
- Both

Should it save progress?
- LocalStorage only (no account needed)
- Google Drive sync (requires auth)
- Simple email-to-resume link

Who is the primary user?
- Community founders starting from scratch
- Existing communities doing a health check or refresher
- Advisors or consultants guiding groups

Three options:
- Full wizard covering all 4 phases (ambitious)
- Phase 1 only from Spark to Gate 1 (realistic MVP)
- Just the checklist generator with document links

CLIPS 5-Layer Model:
- Individual, Community, Intention, Structure, Practice

RCOS 7-Layer Framework:
- Identity and Scope, Membership and Culture, Resources, Governance, Legal, Technology, Operations

Community Alchemy 11 Stages:
- Vision, Recruitment, Governance, Business Models, Infrastructure, Land, Build, Launch, Grow, Mature, Sage

The wizard guides users through these frameworks in sequence, matching each step to the appropriate phase.
