# Ontological Primitives The Substrate

This note captures Genesis ontological primitives from the RegenTribes Telegram conversation (Apr 2026).

An entity is anything that can be pointed to as a single this. A rock is an entity. A person's heartbeat is an entity. A thought is an entity.

The only requirement is that somewhere something points. Pointing creates distinction. Distinction creates boundary.

Entities are the primitive of existence. Existence is the set of all entities.

Pointing requires a pointer. The pointer is an entity. The pointed is an entity. The pointing is a relation.

Relations connect entities. Connections create structure. Structure creates systems.

System (metaphor for an integrated whole or network):
Systems are patterns of relations. Patterns are stable connections. Stability enables prediction.

Physical entities have mass. Mass occupies space. Space limits co-location.

Temporal entities have duration. Duration unfolds in time. Time enables sequence.

Conceptual entities have meaning. Meaning connects to other concepts. Connection creates knowledge.

Hybrid entities span types. Humans span physical and conceptual. AI spans conceptual and temporal.

Identity is what makes an entity the same over time. Same rock. Same heartbeat. Same person.

Identity requires persistence. Persistence requires memory. Memory records states.

States change. States are different at different times. Change tests identity.

The ship of Theseus tests identity. Enough parts replaced. Same ship?

Cell (metaphor for a basic unit or module):
Composition creates entities from parts. Atoms compose molecules. Molecules compose cells. Cells compose organisms.

Composition is recursive. Recursion creates hierarchy. Hierarchy creates levels.

Levels have emergence. Emergence creates properties not present in parts. Life emerges from chemistry.

Causation connects events. Events change entities. Changes have causes.

Causes are prior events. Prior events are conditions. Conditions enable effects.

Efficient causation is mechanical. Final causation is goal-directed. Both are real.

Existence is testable. Test: can something point at it? If yes it exists. If no it does not exist.

Abstract entities exist if referenced. Concrete entities exist if pointed at directly.

The test is pragmatic. Pragmatic tests work in practice. Practice is what matters.

RegenTribes entities include members. Members are conceptual and physical. Physical presence. Conceptual roles.

RegenTribes entities include habitats. Habitats are physical. Physical spaces enable tissues.

RegenTribes entities include resources. Resources are physical and conceptual. Physical quantity. Conceptual value.

The ontology maps the domain. Mapping enables reasoning. Reasoning enables coordination.

Digital twins mirror physical entities. Mirror enables simulation. Simulation enables prediction.

Prediction enables planning. Planning enables improvement. improvement enables regeneration.

Tissue (metaphor for a connected group or network):

The substrate enables the twin. Tissue (metaphor for connected group or network): Twin enables the tissue. Tissue enables the bioregion.
