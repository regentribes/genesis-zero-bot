The prescription is reconciliation. Reconcile with nature. Reconcile with body. Reconcile with each other.

The diagnosis matches. The prescription matches. The frameworks are parallel.

Stage 3 is return. Return to nature. Return to body. Return to community.

The cycle accelerates. The cycle consumes the colonizer.

Balance enables regeneration. Regeneration restores what extraction depleted.

Cities amplify isolation. Proximity without connection. Density without belonging.

Tissue formation enables the return. The return reconnects. Reconnection heals.

Connection restores body. Connection restores nature. Connection restores community.

The fabric torn by extraction. The fabric rewoven by regeneration.
