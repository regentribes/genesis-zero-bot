# Project N.O.M.A.D.-Offline First Knowledge Server

An offline-first knowledge and education server bundle with 8 core capabilities.
N.O.M.A.D. means Offline Mobile Autonomous Device.

Component 1 is Command Center.
Custom UI and API.
Management orchestration.

Component 2 is AI Chat.
Ollama plus Qdrant.
Local LLM plus RAG knowledge base.
RAG means Retrieval Augmented Generation.

Component 3 is Information Library.
Kiwix.
Offline Wikipedia, medical references, ebooks.

Component 4 is Education Platform.
Kolibri.
Khan Academy courses with progress tracking.

Component 5 is Offline Maps.
ProtoMaps.
Downloadable regional maps.

Component 6 is Data Tools.
CyberChef.
Encoding, encryption, analysis.

Component 7 is Notes.
FlatNotes.
Markdown note-taking.

System (metaphor for an integrated whole or network):

Component 8 is System Benchmark.
Built-in.
Hardware scoring and leaderboard.

A Regen package could add tools for regenerative communities.

RegenKnow is an offline permaculture and agriculture knowledge base.
FAO documents, seed saving guides, soil health.

LandMapper is terrain and soil and water cycle mapping for land stewardship.

GovernanceKit has consensus tools, voting, role tracking for community decision-making.

ResourceFlow tracks water, energy, compost loops.
Circular economy.

SeedBank is biodiversity and seed inventory management.

CommunityMesh is local mesh networking info and emergency comms.

Add as a new tool category in the Command Center.
Own Docker Compose service.

Use a lightweight DB for local data.
Static ZIM content for knowledge base.
Simple API endpoints for resource tracking.
