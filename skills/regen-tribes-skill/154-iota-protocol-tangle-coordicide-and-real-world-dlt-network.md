# Iota Protocol Tangle,-Coordicide,-And Real World Dlt Network

System (metaphor for an integrated whole or network):

Ecosystem (metaphor for an interconnected network):

IOTA is a distributed ledger technology designed for the Internet of Ecosystem (metaphor for system or network): Things ecosystem.
Native cryptocurrency is MIOTA.
Total supply is fixed.
Key feature is feeless transactions making microtransactions economically viable.
The IOTA Foundation is headquartered in Berlin with global operations.

Unlike traditional blockchains, IOTA uses a Directed Acyclic Graph structure called the Tangle.
Each new transaction confirms two previous transactions.
Users validate transactions when they issue their own.
This removes the need for miners and their associated fees.
The Tangle becomes faster and more secure as more participants join.

The Tangle replaces linear blockchain with a DAG structure.
Every transaction references two prior transactions.
This creates a web of interconnected validations rather than sequential blocks.
As the Tangle grows, confirmation confidence increases.

Advantages over blockchain:
Feeless transactions enable microtransactions between IoT devices.
Scalability improves as the network grows.
Energy efficiency eliminates energy-intensive mining.
No block size debates since there are no blocks.

The original Tangle required Proof of Work for each transaction.
Users computing two PoW challenges validated the network.
This created a spam resistance mechanism.

Phase 1 is the original Tangle.
2016 launch with full PoW validation.
Coordinator used to secure the network during early stages.

Phase 2 is Chrysalis (IOTA 1.5).
Upgraded in two parts in 2020 and 2021.
Interim step toward full decentralization.
Introduced white-flag mechanism to address conflicts caused by network spam.
Enhanced security, efficiency, and usability.
Many elements necessary for Coordicide were incorporated.

Phase 3 is Stardust.
Mainnet upgrade October 2023.
Transformed IOTA into a multi-asset ledger.
Enabled feeless minting, tokenizing, and transfer of native assets.
Introduced Move models for various output types.
Supports IOTA Smart Contracts and Decentralized Digital Identifiers.
Native tokens and NFTs as first-class citizens on base layer.

Phase 4 is Coordicide and IOTA 2.0.
Project to remove the Coordinator for full decentralization.
IOTA 2.0 represents complete decentralization, leaderless consensus, enhanced scalability.
Public testnet launched Q2 2024.
Introduced Proof-of-Stake consensus replacing PoW.
Slot commitment chains ensure consistent ledger across all nodes.
Starfish BFT protocol developed as improved consensus mechanism.
However, IOTA 2.0 project was abandoned end of 2024 following community vote.

Phase 5 is IOTA Rebased.
Community voted to pivot from IOTA 2.0 to accelerate time-to-market for full decentralization.
Rebased mainnet launched May 2025.
New validator selection process.
Move-based ledger replacing Stardust.
Migration of assets from Stardust ledger.
High-performance Layer 1 network capable of over 50,000 transactions per second.

Mana was a core component of IOTA 2.0 tokenomics.
Mana is a reputation system mitigating Sybil attacks.
It governs access to network resources and ensures fairness.
Mana replaces transaction fees for resource allocation.
Mana is generated by holding IOTA tokens.
Mana can be earned through participation in consensus.
Mana can be received from other Mana holders.

With IOTA Rebased, the security model shifted to enforcing economic incentives and staked tokens.
Validator selection now based on economic disincentives rather than Mana-centric approach.

IOTA Smart Contracts (ISC) enable programmable logic on the IOTA network.
Layer 2 smart contract chains on top of Layer 1 Tangle.
Assembly is the smart contract platform for sharded composable contracts.
Assembly provides feeless and highly scalable base layer for DeFi, NFTs, DAOs.
Shared security across chains.
Trustless interoperability between chains.
Solidity (EVM-compatible) and WebAssembly (Rust, TinyGo, TypeScript) supported.
Users can stake IOTA to earn Assembly (ASMB) tokens.

IOTA EVM released March 2024.
Enables Ethereum-compatible smart contracts on IOTA.
Developers can deploy existing Ethereum tooling and code.

IOTA Streams is a framework for building cryptographic messaging protocols.
Evolution of Masked Authenticated Messaging (MAM).
Allows devices to securely organize, transmit, and retrieve data streams.
Enhanced access control for different message types within a single channel.
Ensures data authenticity and integrity.

IOTA Data Marketplace was launched as proof-of-concept.
Demonstrates how connected devices can securely share and monetize data in real-time.
Zero transaction fees for data exchange.
Unlocks value of unused data from supply chain, smart cities, healthcare.
Open innovation ecosystem for data-driven business models.

IOTA Identity framework allows users to create and manage digital identities.
Strong emphasis on privacy over data harvesting.

No usernames or passwords required.
