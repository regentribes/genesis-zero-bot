# The Unit Physical Domain

The smallest unit is one person with one working skill.

One person with a welding mask and a grinder can make parts.
One person with a 3D printer can make enclosures.
One person with a microcontroller and a sensor can measure the world.

The threshold for contribution is this.
You can contribute when you can make one thing that works.

The threshold is not money.
The threshold is not credentials.
The threshold is capability.
Capability means one working thing.

Tools multiply skill.
One person with power tools equals ten people with hand tools.

The first step is always the same.
Make one thing. Test it. Fix it. Make it again.

A home workshop produces sensor enclosures.
A home workshop produces pipe fittings.
A home workshop produces brackets.
A home workshop produces molds.
A home workshop produces tools for others.

A basic workshop requires weeks to months of collective skilled effort.
The range depends on salvage versus new materials.

Cost depends on new versus used.
Cost depends on specialization.
Start cheap. Upgrade as needed.

The time to first output ranges from one day to one week.
First output builds confidence.
First output reveals the next step.

No permission is needed.
No supplier is needed.
No license is needed.
Start where you are.

Steel covers most structural needs.
Plastic covers most enclosure needs.
Wood covers most formwork needs.
Concrete covers most foundation needs.
These four materials cover 90 percent of needs.

Learn one process per month.
Welding. Casting. 3D printing. Machining.
Start with what you have.
Move to what you need.

Community starts when one person shows another.
Teaching by making is the method.
The demo is the lesson.
Words are secondary.
