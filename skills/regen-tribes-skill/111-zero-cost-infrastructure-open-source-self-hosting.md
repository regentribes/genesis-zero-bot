# Zero Cost Infrastructure Open Source Self Hosting

This note captures the zero-cost infrastructure approach for regenerative communities.

Zero cost means no vendor lock-in. No recurring fees. No surveillance.

Self-hosting means community controls the infrastructure. Community owns the data. Community owns the computation.

Open source means the code is public. Public code is auditable. Auditable code is trustworthy.

The combination enables sovereignty. Sovereignty enables independence.

Single board computers run everything. Raspberry Pi provides GPIO. GPIO connects sensors.

Used servers provide more power. Used Dell PowerEdge costs $200. PowerEdge runs virtual machines.

Refurbished ThinkPads run $100. ThinkPad runs Linux. Linux runs everything.

Hardware under $500 covers most needs. Needs include storage. Needs include computation. Needs include networking.

Debian is stable. Debian updates slowly. Slow updates mean stability.

 NixOS is reproducible. NixOS configuration is code. Code is version controlled.

 Fedora is modern. Fedora includes latest packages. Latest packages include newest features.

 Alpine is minimal. Alpine uses musl libc. Musl enables containers.

Nextcloud provides file sync. File sync replaces Google Drive. Self-hosting preserves privacy.

Gitea provides Git. Git enables code collaboration. Self-hosting preserves code.

SurrealDB provides database. Database replaces proprietary cloud. Self-hosting preserves data.

Navidrome provides music. Music streams to devices. Self-hosting preserves media.

Cloudflare Tunnel provides connectivity. Connectivity without open ports. Open ports are security risk.

Tailscale provides VPN. VPN enables remote access. Remote access enables work.

Pi-hole provides DNS filtering. DNS filtering blocks ads. Blocking ads preserves bandwidth.

Backups run nightly. Nightly backups use rsync. Rsync syncs to external drive.

External drives rotate weekly. Weekly rotation provides version history. Version history enables recovery.

Off-site backup uses rsync.net. rsync.net provides encrypted storage. Encrypted storage preserves confidentiality.

Hardware costs $200-500 one-time. This includes server and storage.

Electricity costs $10-30 per month. This powers the server.

Domain costs $10 per year. This provides human-readable addresses.

Total cost is under $50 per month. This is the minimum viable infrastructure.

Updates apply automatically. Automatic updates prevent exploits. Exploits compromise systems.

Firewalls block unused ports. Unused ports are attack surface. Blocking surface prevents intrusion.

Fail2ban blocks brute force. Brute force attempts are common. Blocking attempts preserves access.

Self-hosted infrastructure enables sovereignty. Sovereignty enables community control.

Control means data stays local. Local means no surveillance. No surveillance means freedom.

The infrastructure must be maintainable. Maintainability requires documentation. Documentation requires investment.
