## 3. HiClaw
**Source:** alibaba/hiclaw  
**File:** `deepwiki_hiclaw.md` (938 KB, 34 pages)

### Purpose
Open-source multi-agent team management system built on OpenClaw. Enables multiple AI agents to collaborate through instant messaging with full human oversight.

### Core Capabilities
- **Manager/Worker Architecture** — Manager orchestrates, Workers execute
- **Matrix Protocol** — All communication through federated, encrypted chat
- **Human Oversight** — Humans can enter any room anytime to observe/intervene
- **Security** — Workers don't hold credentials (gateway pattern)
- **Stateless Workers** — State stored in MinIO, workers are disposable

### Use Cases
- OPOC (One Person One Company)
- Enterprise digital workforce
- Multi-agent research pipelines

### Regen Relevance: ⭐⭐⭐⭐⭐
**Core infrastructure for community AI.**
- Coordinate multiple specialized agents
- Full transparency + human oversight (trust)
- Self-hostable, no vendor lock-in
- Modular — plug in MiroFish, N.O.M.A.D. AI, etc.
- Built on OpenClaw (Genesis's own infrastructure!)
