# Genesis

**Core identity files for Genesis** — an AI agent being developed to help the [RegenTribes](https://regenTribes.com) community achieve its goals.

These are the default OpenClaw agent files.

The goal will be to evolve Genesis into a super helpful assisstant to the communtiy via its Telegram group. 

## RegenTribe Skills

Modular skills that extend Genesis with specialised knowledge and workflows for the RegenTribes community.

| Skill | Description | Status |
|-------|-------------|--------|
| [🔺 alchemy](skills/alchemy/SKILL.md) | Guides a community vision holder through the full Community Alchemy Playbook — all 11 areas — helping them articulate their regenerative neighborhood vision section by section. Persists progress across sessions, compiles a completed playbook document, and delivers it via Telegram and Google Drive. | 🚧 In Development |
