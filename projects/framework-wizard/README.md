# Framework Wizard

**Status:** Draft
**Concept:** A wizard-based web app that walks users through the anti-capture framework to create a regenerative neighborhood plan.

---

## Vision

A question-by-question journey that:
1. Asks users questions from the framework journey
2. Fills in a shareable community plan
3. Generates key documents every community needs
4. Project-manages the build/operate/grow phases

---

## Features (Draft)

- Multi-step wizard (Vision → Community → Land → Finance → Timeline)
- Progress tracking
- Auto-save
- Document generation (PDF/export)
- Shareable output
- Project management (milestones, tasks, reminders)
