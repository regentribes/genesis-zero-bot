## Project Overview

**Name:** Framework Wizard
**Purpose:** Build a comprehensive anti-capture design framework for RegenTribes that maps theoretical capture patterns to practical regenerative neighborhood systems.
**Origin:** Synthesized from conversation with Airic Easm (2026-03-26)

---

## Scope

### In Scope
- Document the theory of capture (4 pillars: Capital, Narrative, Identity, Governance)
- Define anti-capture design principles
- Map principles to regenerative neighborhood mechanics
- Identify failure modes and mitigations

### Out of Scope
- Full implementation of any specific system
- Legal/tax structure design (Phase 2)
- Community deployment (Phase 3)

---

## Deliverables

| # | Deliverable | Description |
|---|-------------|-------------|
| D1 | Anti-Capture Framework Doc | Core theory + principles (created) |
| D2 | Design Principles Detail | Expand each principle into mechanics |
| D3 | Regenerative Neighborhood Map | How framework applies to RN systems |
| D4 | Failure Mode Analysis | Where it breaks + mitigations |
| D5 | Project Roadmap | Phased build plan with milestones |
