# Skills Archive

Skills in this directory are NOT loaded into Genesis context.

## Why Archived?

Skills are archived when they are:
- Specialized (only needed for specific projects)
- Large (contain extensive documentation)
- Rarely used in current RegenTribes workflow

## Restoration

To restore a skill to active use:
```bash
mv skills-archive/<skill-name> skills/<skill-name>
```
