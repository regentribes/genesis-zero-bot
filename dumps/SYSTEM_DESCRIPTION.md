## User Model

### Four Types of User Statements

The system separates user input into four groups. This separation is mandatory.

#### Type One: Need Statements

Need statements describe personal requirements. Need statements answer: What do I require to survive? Need statements answer: What do I require to feel safe? Need statements are individual. Need statements do not require other people.

#### Type Two: Belief Statements

Belief statements describe mental models. Belief statements answer: What do I think is true? Belief statements are personal opinions. Belief statements may be wrong.

#### Type Three: Principle Statements

Principle statements describe operating rules. Principle statements answer: What rules do I follow? Principle statements guide personal behavior.

#### Type Four: Value Statements

Value statements describe relational commitments. Value statements answer: What do I practice with others? Value statements require other people. One person cannot practice values alone. Generosity requires a receiver.
